## How to Run

1. Clone the repository:

```
$ git clone https://github.com/Dumbledore-on-Strive/goodreads-app.git
$ cd goodreads-app
```

2. Install dependencies:

```
$ pip install -r requirements.txt
```

3. Start the application:

```
streamlit run streamlit_app.py
```
